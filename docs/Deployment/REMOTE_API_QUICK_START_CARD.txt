┌──────────────────────────────────────────────────────────────────────────┐
│ 🔧 TROUBLESHOOTING                                                      │
└──────────────────────────────────────────────────────────────────────────┘

   Problem: Connection lost after sleep
   → In Bitburner: Options → Remote API → Connect
   → Server stays running, just reconnect!

   Problem: Forgot to start server
   → npm run watch:remote (in bitburner-remote-api folder)
   → Reconnect in Bitburner

   Problem: File not syncing
   → Check PowerShell for "[watch:remote] file.js changed"
   → Verify "Connected" status in Bitburner
   → Try save again (Ctrl+S)

   Problem: TypeScript errors in console
   → Ignore them! JavaScript files work perfectly!
   → Those are just type definition warnings
